### Потребность в приложении: ###
* с 2018 резко увеличилось количество продаж "холодными звонками" с незнакомых номеров
* первопричина - организации стали использовать возможность - если иметь пул телефонных номеров масштаба нескольких десятков-сотен-тысяч, то, даже если пользователи будут блокировать номера, всегда остается возможность подставить другой номер
* последствия - если звонки приходят в любое время, и нет возможности их заблокировать, то это становится отвлекающим и раздражающим фактором
### Цели работ:
- установить на смартфон под ОС Андроид приложение, блокирующее пул телефонных номеров любого масштаба целиком
### Полученные результаты:
- приложение Xamarin.Android для установки в качестве приложения телефона по умолчанию
- основное окно с кнопками и полем вывода сообщению пользователю о состоянии
- окно приема телефонного вызова с кнопками "поднять/опустить трубку" и "отключить микрофон"
- таблица Google Sheet с перечнем паттернов регулярных выражений заблокированных пулов номеров
- модуль чтения и записи таблиц Google Sheet
- модуль проверки регулярными выражениями номера входящего звонка
- модуль оповещения пользователя:
    - если номер телефона соответствует регулярному выражению, то смартфон переводится в вибро-режим
    - иначе, пользователю показывается push-уведомление для открытия окна приема входящего вызова
- модуль выгрузки истории заблокированных звонков в отдельную таблицу Google Sheet
